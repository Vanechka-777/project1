# Калькулятор расхода топлива

Простое и удобное приложение для расчета расхода топлива автомобиля, написанное на Python с использованием PyQt5 и SQLite.

# Описание
Этот калькулятор позволяет:
1. Рассчитать расход топлива на 100 км
2. Определить стоимость поездки
3. Узнать стоимость 1 км пути
4. Сохранять историю расчетов в базу данных
5. Просматривать и экспортировать историю расчетов

# Основные возможности:
1. Расчет расхода топлива - автоматический расчет по введенным данным
2. Сохранение истории - все расчеты сохраняются в локальную базу данных
3. Просмотр истории - таблица с всеми предыдущими расчетами
4. Экспорт данных - возможность экспорта истории в текстовый файл
5. Очистка истории - удаление всех сохраненных записей

# Расчетные показатели:
1. Расход топлива (л/100км)
2. Общая стоимость поездки (руб)
3. Стоимость 1 км пути (руб/км)

# Требования:
Python 3.9
PyQt5
SQLite3 (входит в стандартную поставку Python)
